# AI Puzzle Generator Setup

## Get Free Google Gemini API Key

1. **Go to Google AI Studio**
   - Visit: https://makersuite.google.com/app/apikey
   - Sign in with your Google account

2. **Create API Key**
   - Click "Create API Key"
   - Select "Create API key in new project" or use existing project
   - Copy the API key

3. **Add to Render Environment Variables**
   - Go to Render dashboard: https://dashboard.render.com
   - Click on your "brainboost-16cb" service
   - Go to "Environment" tab
   - Click "Add Environment Variable"
   - Name: `GEMINI_API_KEY`
   - Value: (paste your API key)
   - Click "Save Changes"

4. **Service will auto-redeploy** (wait 2-3 minutes)

## Features

- **AI-Generated Puzzles**: Each puzzle is unique and generated by Google's Gemini AI
- **Difficulty Levels**: Easy, Medium, Hard
- **Free Tier**: 60 requests per minute (more than enough)
- **Smart Fallback**: If AI fails, uses backup puzzles

## API Limits (Free Tier)

- 60 requests per minute
- 1,500 requests per day
- Completely FREE forever

## Test Your Setup

After adding the API key, test the puzzle generator:
1. Go to your app: https://brainboost-eight.vercel.app
2. Navigate to Puzzles section
3. Select difficulty and click "Generate Puzzle"
4. You should see AI-generated puzzles!
